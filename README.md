# Гиф-сервис с зависимостью от валют

Сервис сравнивает состояние валюты за сегодня и вчера и отдаёт случайный GIF в зависимости от скачка
или падения валюты


-------------------------------------------------

**Минимальные требования:**

- Java 17+
- Lombok
- Feign
- Spring Web (стартер)
- Maven 4.0+

-------------------------------------------------

**Внешние API:**

- [Open Exchange Rates] (https://openexchangerates.org/)
- [Giphy] (https://giphy.com/)
